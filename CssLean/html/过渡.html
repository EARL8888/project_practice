如需向多个样式添加过渡效果，请添加多个属性，用逗号隔开：

向宽度、高度和转换添加过渡效果

注释：本例在 Internet Explorer 中无效。

所有的转换属性：

1. transition-property: 规定应用过渡的 CSS 属性的名称。

2. transition-duration: 定义过渡效果花费的时间。默认是 0。

3. transition-timing-function: 规定过渡效果的时间曲线。默认是 "ease"。(匀-速、慢-快-慢、慢-快、快-慢)

4. transition-delay: 规定过渡效果何时开始。默认是 0。
